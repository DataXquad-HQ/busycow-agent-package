# Package Map

## The two setup classes

### 1. Infrastructure setup
This is everything that must exist before any agent is installed:
- Hermes runtime conventions
- contextual layer
- collaboration layer
- governance and verification

Start at:
- `playbooks/infrastructure/README.md`
- `artifacts/infrastructure/README.md`

### 2. Agent setup
This is one installable package per agent:
- `SOUL.md`
- skills
- workspace harness
- setup instructions
- verification path

Start at:
- `playbooks/agents/README.md`
- `artifacts/agents/`
