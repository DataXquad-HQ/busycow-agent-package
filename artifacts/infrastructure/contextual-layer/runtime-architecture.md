# Contextual Layer Runtime Architecture

Use this as the practical architecture reference for installation.

## Core split

| Layer | Role |
|---|---|
| GBrain canonical | approved durable knowledge |
| GBrain evidence | source trail and what happened |
| Hindsight | episodic and role-scoped memory |
| structured systems | current operational truth |
| workspace | drafts, runbooks, queues, current operating context |

## Install rule
Do not collapse these into one storage surface.

## Reading priority
1. explicit human instruction
2. structured current-state system
3. canonical knowledge
4. approved decision record
5. workspace operating docs
6. episodic memory
7. inference
