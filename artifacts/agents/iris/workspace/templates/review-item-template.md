# Review Item Template

Use when a candidate must be staged for governance instead of being published directly.

## Copyable skeleton

```md
# Review Item

- title:
- type: conflict | promotion_candidate | policy_change | stale_context | approval_needed
- current_source:
- why_it_matters:
- recommended_action: review_now | hold | reject | gather_more_evidence | escalate
- owner_or_approver:
- evidence_links:

Optional:
- destination_if_approved:
- deadline_or_review_time:
- blocking_impact:
- candidate_id:
- source_event_id:
```

## Minimum quality bar
- owner/approver is named
- evidence exists
- destination is explicit if approval would trigger publication
- rationale explains business consequence, not just documentation hygiene
