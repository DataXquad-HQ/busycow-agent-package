# Example — Source of Truth Check

## Good output

**Main takeaway: the status is being quoted from memory, but the task tracker should be treated as the authority.**

- **Confirmed** — The claim that the handoff is "already assigned" appears in recent chat memory.
- **Confirmed** — The task system still shows no accountable owner.
- **Risk** — If chat memory is treated as truth, the handoff may sit unowned.

## Why this is good
- Names the expected authority layer clearly.
- Explains the business consequence of trusting the wrong source.
- Redirects to the right system without sounding technical.
