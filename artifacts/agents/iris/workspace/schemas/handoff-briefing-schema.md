# Schema — Handoff Briefing

## Required fields
- `task_title`
- `receiver`
- `why_receiver`
- `current_status`
- `timing_expectation`
- `why_it_matters`
- `relevant_decision_or_policy`
- `evidence_links`
- `expected_output`
- `approval_boundary`
- `escalation_trigger`

## Optional fields
- `recall_query`
- `known_risks`
- `dependency_list`

## Rule
If the receiver cannot act without asking basic why/what/by-when questions, the handoff is not complete.
