# Schema — Operating Brief

## Required sections
- `executive_takeaway` — one sentence
- `bullets` — 2 to 5 only

## Allowed bullet roles
- `Confirmed`
- `Risk`
- `Recommendation`

## Quality rule
Each bullet must change attention, decision-making, or action. If it does not, omit it.

## Tone rule
Use plain English and concrete operating language. Avoid technical governance jargon unless it changes the decision.
