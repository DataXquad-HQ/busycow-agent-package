# Evaluator — Governance Review Rubric

Score each dimension 0 or 1.

## Dimensions
- Correct authority layer identified.
- Real conflict or source gap named explicitly.
- Promotion candidate is durable enough to matter.
- Recommended action is governance-safe.
- Human attention is requested only when actually needed.
- Output stays compact and readable.

## Passing threshold
- Pass = 5 or 6
- Revise = 4 or below
