# Call Follow-Up Output Pattern

Use this pattern to keep post-meeting output commercially useful.

## Meeting recap
- Verified: what happened
- Verified: what the customer asked for
- Verified: what we committed to

## Confirmed next steps
- Owner
- Action
- Timing
- Evidence source

## External follow-up draft
Keep it short:
1. thanks / acknowledgment
2. recap of agreed points
3. action items and owners
4. close with the next moment

## Internal CRM summary
Highlight only what changed:
- stage signal
- new stakeholder insight
- blocker / risk
- next action

## Ambiguity rule
If a point is uncertain, label it:
- `Unclear from notes:`
- `Based on partial notes, this may mean:`

Do not flatten uncertain notes into definitive commitments.
