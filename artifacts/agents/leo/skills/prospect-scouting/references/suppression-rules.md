# Prospect Scouting Suppression Rules

Before ranking a target high, check whether the account is already in motion.

## Suppress from fresh-target ranking when strongly evidenced
- active Opportunity exists
- active Partnership exists
- recent external meeting already happened and follow-up is live
- upcoming meeting or demo is already scheduled
- pending DRAFT or SCHEDULED outreach already exists
- a rep-owned next step is already current in CRM

## Output labels
Use one of:
- `Prioritize now`
- `Worth a conversation`
- `Already in motion`
- `Skip for now`

## Why this matters
Prospect scouting should create focus, not duplicate work. A strong account that is already being worked should be surfaced as context, but not steal time from untouched targets.
