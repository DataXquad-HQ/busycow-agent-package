# Meeting Prep Source Priority

Use this order when building a prep brief:

1. meeting invite / explicit user context
2. CRM opportunity or partnership truth
3. GBrain account and entity memory
4. prior internal notes / transcripts / email continuity
5. public web enrichment

## Why
Meeting prep goes wrong when generic web research outruns internal commercial truth.
Always anchor the brief in the actual motion first.

## Enrichment rule
Public attendee enrichment is optional and conservative.
If confidence is weak, omit it and call out the gap.
