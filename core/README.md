# Core

Hermes Agent runtime setup, identity templates, and core skills.
**This layer must be installed before anything else.**

## Contents

| Path | What |
|------|------|
| `SETUP.md` | Step-by-step Hermes install + initial configuration |
| `templates/` | Starter templates for SOUL.md, MEMORY.md, USER.md |

## Install order

1. `SETUP.md` — Hermes install, Lark MCP, GBrain init
2. Fill in templates from `templates/` to create your identity files
3. Proceed to `../third-party-tools/` → `../shared-skills/` → `../agent-teams/`
